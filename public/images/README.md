# Images Directory

This directory contains images for the Anniversary Gallery demo.

## Required Images

The gallery expects the following images:
- `1.jpg` - Lễ khai giảng năm 1980
- `2.jpg` - Lễ tốt nghiệp 1995
- `3.jpg` - Kỷ niệm 30 năm thành lập
- `4.jpg` - Sinh viên xuất sắc 2005
- `5.jpg` - Hoạt động thiện nguyện
- `6.jpg` - Lễ kỷ niệm 50 năm

## Adding Images

To add your own images:
1. Place your images in this directory
2. Name them `1.jpg`, `2.jpg`, `3.jpg`, etc.
3. Or update the image paths in `components/anniversary-gallery.tsx`

## Using Placeholder Images

If you don't have images yet, you can use placeholder services:
- https://placehold.co/800x600
- https://picsum.photos/800/600
- Or use existing images from the public folder
